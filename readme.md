For running this web application you will need to do several steps:

0) clone the repository (git clone / wget)
1) install nodejs (brew install node)
2) install elasticsearch (brew install elasticsearch)
3) from terminal go to di
4) from the terminal cd into the "e-commerce-course-project"
5) from the terminal run "npm init"
6) from the terminal run "elasticsearch" (OSX users)
7) from the terminal in a new tab in the same location "e-commerce-course-project" run node server.js
8) open this URL http://localhost:3000 from the browser
